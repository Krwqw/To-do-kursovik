DTO (Data Transfer Object) — это простой класс, который содержит только данные (без методов, без логики). 
Его единственная задача: безопасно и аккуратно переносить информацию между клиентом (фронтендом) и сервером (бэкендом).


[Required] - поле обязательно
[EmailAddress] - проверяет формат лялял@лялля.лялял
