# TaskBoard

Курсовой проект: Ковалева Ульяна, Григорьева Надежда, группа 3-1 ИС.

## Запуск

1. Открыть `Kursovichok2.sln` в Visual Studio.
2. Запустить проект `Kursovichok2` без отладки.
3. Если браузер не открылся автоматически, перейти по адресу `http://localhost:5272`.
   При запуске HTTPS-профиля адрес будет `https://localhost:7029`.

Фронтенд находится в папке `kirs_ulyana` и при запуске backend-проекта отдаётся автоматически.
